NPC 이름 : 레오나
스토리 : 레오나는 과거 용병으로 일하다가 기사가 되어 왕국을 지키며 무사히 살아남았다. 이제는 옛날 동료들과 함께 왕국을 지키며 평화롭게 살고 있다.

퀘스트 1: 불의 징표
이름 : 불의 징표
내용 : 최근 마을 주변에 불이 붙어 인명 피해가 발생했다. 불길의 근원인 불의 징표를 찾아서 없애야 한다.
태스크 : 불의 징표 x3 제거
보상 : 경험치 500, 청동 검 x1, 100골드

퀘스트 2: 타락한 사제
이름 : 타락한 사제
내용 : 요새 근처 사원에서 타락한 사제가 출몰하면서 주민들이 위험해졌다. 타락한 사제를 제거하여 주민들을 보호해야 한다.
태스크 : 타락한 사제 x1 제거
보상 : 경험치 800, 심판의 검 x1, 200골드

퀘스트 3: 마법진 수호
이름 : 마법진 수호
내용 : 마을 근처 마법진에서 괴물들이 자주 출몰하여 주민들이 위험해졌다. 마법진을 수호하여 주민들의 생명을 지켜야 한다.
태스크 : 마법진 수호 x5
보상 : 경험치 1000, 힘의 반지 x1, 300골드

퀘스트 4: 드래곤의 눈물
이름 : 드래곤의 눈물
내용 : 전설 속 드래곤의 눈물을 얻어서 옛날 동료의 아들에게 선물로 주려 한다. 하지만 드래곤의 눈물은 가장 강력한 드래곤인 파이어 드래곤의 눈물이다. 파이어 드래곤의 눈물을 얻기 위해 파이어 드래곤을 사냥해야 한다.
태스크 : 파이어 드래곤 x1 사냥 후 드래곤의 눈물 획득
보상 : 경험치 1500, 드래곤의 검 x1, 500골드
